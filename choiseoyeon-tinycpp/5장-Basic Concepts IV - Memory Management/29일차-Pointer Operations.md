- 포인터는 메모리의 위치를 가리키는 값이다
- 포인터 접근은 포인터가 가리키는 위치에 저장된 값을 가져오는 것이다
- subscript 연산자는 주어진 위치의 원소에 접근 하도록 한다.
- 포인터의 타입은 아키텍쳐에 따라 32/64비트 부호 없는 정수이다
- + - ++ -- 비교 연산자, [], * 연산자만 사용 가능하다
- 포인터는 명시적으로만 정수형으로 형변환이 가능하다
```
// 참조
int* ptr1 = new int;
&ptr1 = 4;
int a = *ptr1

// 배열 접근
int* ptr2 = new int[10];
ptr2[2] = 3;
int var = ptr2[4];

// 흔한 오류
int* ptr1, ptr2; // 포인터 하나에 변수 하나
int* ptr1, *ptr2;// 포인터 개
```
- `ptr[1]` == `*(ptr+1)` 이다
