## C++의 타입 종류
- long double도 제공한다
- C++23 버전 이전에는 덜 정확한 부동소수점이 제공되었따.
	- 몇몇의 컴파일러들에서는 half를 지원하기도 한다.
	- OpenGL, Photoshop, Lightroom 등에서 지원
- C++에서는 128 비트 짜리 정수 타입을 지원하지 않는다.
## void 타입
- void 타입은 값 없이는 정의 되지 않은, 즉 완전하지 못한 타입이다.
- 또한 반환 값이나 파라미터가 없는 함수를 뜻하기도 하다
- C에서는 `sizeof(void)`가 1로 나오지만, C++에서는 `sizeof(void)`가 컴파일 되지 않는다.
## nulptr 키워드
- C++11에서는 모든 null 포인터와 NULL 메크로를 대체할 목적으로 `nullptr`키워드를 제공했다.
- nullptr은 nullptr_t 의 오브젝트이다.
