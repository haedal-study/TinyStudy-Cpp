## Promotion(확대)
- 작은 데이터 타입을 큰 데이터 타입으로 변환하는 것
- 부호 확장이 일어날 수 있음
```
int16_t x = -1;

int y = x; // 부호 확장

cout << y; // -1 출력
```
## Truncation(축소)
- 큰 데이터 타입을 작은 데이터 타입으로 변환하는 것
- 작은 데이터 타입이 표현할 수 있는 범위를 벗어난 값은 모드 연산으로 나머지 값이 저장됨
```
int x = 65537; // 2ˆ16 + 1

int16_t y = x; // x % 2ˆ16

cout << y; // 1 출력

int z = 32769; // 2ˆ15 + 1

int16_t w = z; // (int16_t) (x % 2ˆ16 = 32769)

cout << w; // 오버플로우 발생, -32767 출력
```
