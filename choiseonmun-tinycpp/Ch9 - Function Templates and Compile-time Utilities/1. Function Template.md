# 메모
## 개요
- 오버로딩이 가능하다.
```cpp
template <typename T>
T add(T a, T b) { return a + b;}

template <typename T>
T add(T a, T b, T c) { return a + b + c;}

template <int C, typename T>
T add(T a, T b) { return a + b + c; }
```
- 함수 템플릿은 전체 특수화만 가능하다.
