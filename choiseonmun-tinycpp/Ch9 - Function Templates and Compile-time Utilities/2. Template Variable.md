## 개요
- C++14부터 제공
```cpp
template <typename T>
constexpr T pi{ 3.1415926535897932385 };

template <typename T>
T circular_area(T r)
{
	return pi<T> * r * r;
}

circular_area(3.3f); // float
circular_area(3.3); // double
circular_area(3); // [compile error] narrowing
```
