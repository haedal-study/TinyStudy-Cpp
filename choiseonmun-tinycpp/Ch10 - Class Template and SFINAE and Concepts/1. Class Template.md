# 메모
## 개요
- 클래스에도 템플릿을 적용할 수 있다.
- 클래스는 부분 특수화가 가능하다.
	- 모든 클래스 특수화는 완전히 새로운 클래스며, 제네릭 클래스와 아무것도 공유하지 않는다.
```cpp
// 전부 다른 클래스다.
template <typename T, typename R>
struct A {}; // generic class template

template <typename T>
struct A<T, int> { }; // partial specialization

template <>
struct A<float, int> {}; // full specialization
```
