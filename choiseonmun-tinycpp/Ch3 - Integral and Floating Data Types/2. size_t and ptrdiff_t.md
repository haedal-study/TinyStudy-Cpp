# size_t
- 현재 아키텍처에서 얼마나 데이터를 많이 저장할 수 있는지를 나타내는 타입
- 부호 없는 정수 타입이다.
- `sizeof()`의 반환 타입이다.
- 크기 측정을 표현할 때 사용한다.
- 32비트 아키텍처에서는 4바이트, 64비트 아키텍처에서는 8바이트다.
- `uz` / `UZ` 리터럴(since C++23)

# ptrdiff_t
- 현재 아키텍처에서 얼마나 데이터를 많이 저장할 수 있는지를 나타내는 타입
- 부호 있는 정수 타입이다.
- 포인터 뺄셈을 계산할 때 사용한다.
- 32비트 아키텍처에서는 4바이트, 64비트 아키텍처에서는 8바이트다.
- `z` / `Z` 리터럴(since C++23)
